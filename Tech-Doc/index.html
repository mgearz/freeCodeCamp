Introduction

Learning to properly organize and name your video files is crucial for the best Plex experience. After all, it's Plex's built-in scanners that do all the heavy-lifting for you, automatically matching all the metadata. It's what makes Plex a top choice for Movie/TV Show fans alike, giving you your own personal Netflix-like experience. (Minus the pesky content deletion)

But it does have its limits, and if you don't take the time to properly name & organize your files, you'll be prone to mismatches and errors. And then you'll be stuck trying to manually match them, or even worse, having to fill out all the details one-by-one.

NOTE: This guide is for file naming purposes only and assumes you have already set-up your Plex Media Server.

Feel free to use the Navigation Bar to jump to whichever section applies to you. Let's get started!

TV Shows

TV Shows should be kept in a folder called "TV Shows", pretty self-explanatory huh? I mean, technically you can name it whatever you want as long as you point Plex to it, but the point is, it's best practice to separate it from your Movies and other media types.

Then, within that folder you create a sub-folder for each show's name, followed by another sub-folder for each season.

SYNTAX: /TV Shows /Show Name (Year) /Season #
EXAMPLE: /TV Shows /The Office (2005) /Season 01

Each individual video file should then be placed within its appropriate folder (by season) and named using the syntax below:

SYNTAX: /Show Name (Year) - s##e##.filetype
EXAMPLE: /The Office (2005) - s01e01.mp4

(Note: s## = Season Number; e## = Episode Number)

Movies

Movies should be kept in their own folder called "Movies". This helps Plex identify the media type vs other formats such as TV Shows.

Unlike TV Shows, you can just place the movie file within this folder directly. The exception being if your movie is in Multiple Parts or has Extra Features.

Option 1 (Movie Only)

SYNTAX: /Movies /Movie Name (Year).filetype
EXAMPLE: /Movies /Tommy Boy (1995).mp4

Option 2 (Movie + Extra Features)

SYNTAX: /Movies /Movie Name (Year) /Movie Name (Year).filetype + Extra Features files
EXAMPLE: /Movies /Tommy Boy (1995) /Tommy Boy (1995).mp4 , Theatrical Trailer-trailer.mp4 ...
Multi-Part Movies

Sometimes movies are split into multiple parts. Usually this is due to file size, particularly for Blu-Ray Rips and 4K+ video. Instead of massive 50GB files, that some hard-drives cannot handle, it is split into multiple smaller files to make them more manageable.

Luckily, Plex has a built-in way to play multi-part movies seamlessly. All you need to do is organize and name them properly:

SYNTAX: /Movies /Movie Name (Year) /Movie Name (Year)-pt#.filetype , Movie Name (Year)-pt#.filetype ...
EXAMPLE: /Movies /Alien (1979) /Alien (1979)-pt1.mp4 , Alien (1979)-pt2.mp4 ...
Extra Features

A lot home video releases (DVD/Blu-Ray) included Extra Features to complement the Main Feature. For us cinephiles, Plex allows us to display those right alongside our Movies.

SYNTAX: Descriptive Name -ExtraType.filetype

Extra Types

  • Behind the Scenes (-behindthescenes)
  • Deleted Scenes (-deleted)
  • Featurettes (-featurette)
  • Interviews (-interview)
  • Short Films (-short)
  • Trailers/Previews (-trailer)
  • Other Videos (-other)
EXAMPLE: /Movies /Spaceballs (1987) /Spaceballs The Documentary-featurette.mp4 , In Conversation with Mel Brooks-interview.mp4 ...
